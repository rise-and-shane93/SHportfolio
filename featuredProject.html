Adena Harper, Mary Kay Cosmetics

This is my first web development project! My mother, Adena Harper, wanted a website for her own Mary Kay® unit so she could bring in new clientele. Since this was my first website, I utilized the beginner knowledge I had acquired from to create it!

View website!

My mother and I discussed the following goals for the website:

  1. Have a colorful theme that matches Mary Kay®.
  2. Showcase monthly specials on products.
  3. Let prospective clients request appointments and place orders.
  4. Post the events for the current month.

Brainstorming!

Mom first suggested building the site using GoDaddy's web templates. However, that would not help me hone my web development skills so I decided to design the site from HTML and CSS. Click on any of the three photos to see an example.

even with captions...

Based on what I observed in the templates, I knew that I wanted to incorporate a light and appealing color theme as well as easy to navigate for users. Therefore, I wanted to ensure that the page was easy to update for future product specials and events.

Customization

To elaborate on an earlier point, I applied the lessons I learned from TreeHouse thus far to build this website. Specifically, I utilized HTML5, CSS3, and Bootstrap 4. Therefore, I had to heavily rely on those languages to customize the website.

However, I was satisfied in what I was able to accomplish with that entry-level skill set. Not only was I able to style and organize the page to make it presentable to users, but I also added personal customization to make the user experience enjoyable!
